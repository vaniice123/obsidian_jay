# 장내 미생물 불균형 — 디스바이오시스(Dysbiosis)

디스바이오시스(dysbiosis)는 장내 미생물 생태계의 균형이 무너진 상태를 의미하며, 유익균 감소, 유해균 증가, 또는 전체 다양성 저하로 특징지어진다. 마이크로바이옴 관련 거의 모든 질환의 공통 기저 메커니즘이다.

## 유형
1. **유익균 감소**: Bifidobacterium, Lactobacillus, Faecalibacterium prausnitzii 등의 감소
2. **유해균 증가**: 병원성 세균의 과증식
3. **다양성 저하**: 전체 미생물 종의 다양성 감소

## 원인
- 항생제 사용 (가장 흔한 원인)
- 서구화 식이 (고지방, 저섬유질)
- 스트레스, 수면 부족
- 노화
- 감염, 만성 질환

## 결과
- 장벽 기능 약화 ("leaky gut") → 내독소(LPS) 혈류 유입
- 전신 저등급 만성 염증
- [[Microbiome-KB/wiki/short-chain-fatty-acids\|short-chain-fatty-acids]] 생산 감소
- 면역 조절 이상

## 관련 질환

- 신경: 우울증, 파킨슨, 알츠하이머 → [[Microbiome-KB/wiki/gut-brain-axis\|gut-brain-axis]]
